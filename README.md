### Installing SCEO

To install **SCEO**, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/Yeabtsega-Tesfaye/sceo.git
```

Windows:

```bash
git clone https://github.com/Yeabtsega-Tesfaye/sceo.git
```
